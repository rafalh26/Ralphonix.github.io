Model 1: Circuit Board


I used Datasheet provided by Micron for reference - not commercial private use
Micron - Source Site
Here I made an model of simple cuircuit board using simple tools such as extrusion extruded cut and some basic scatches.
I started from learning what the satches are and what shapes I can create.
Then using these I extruded the scatch and made a simple DDR RAM cuircuit Board.
A Circuit Boards can be used later on for any assembly and further parts references.
In the end I learned about some basics Apearence tools and how I can utilize that to add some real look of an object.

model 2: Circuit Board
Here I managed to learn what are 3D scatches for and how to use these.
After I had made 3d scatch then I used sweep feature to make the solid that I wanted for a cable.
Crucial here was to make tangent relation to a construction vertical line at the edges where the cable connect to other object in my case the cable plug.
The Plug itself was a basic extrude feature with some fillets at the end.
