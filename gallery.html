Model 1: Circuit Board


I used Datasheet provided by Micron for reference - not commercial private use
Micron - Source Site
Above a simple model to represent cuircuit board using simple tools such as Extrude and Extruded Cut Features and some basic scatches.
I started from learning what the scatches are and what shapes I can create.
Thanks to provided dimmensions from the Micron Webside data sheet I could make a scatch to extrude basic shape
Then after extruded basic shape I could add more scatches to make model more detailed.
These extra scatches were then used to make Extruded Cut and further Extrude Features.
Thanks to using these simple features I finaly got the shape that I wanted to achieve.

As a finishing touch I simply added some Appearance for the model to look more realistic and made a rendering using PhotoView 360
A Circuit Boards can be used later on for any assembly and further parts references.
But the most important it gave me a good basic understanding of scatches and Extrude / Extrude Cut Features.

Earpods Headphones
Above a simple model to represent Usage of Sweep Feature.
I started from learning what the 3d scatches are and how use use em for a simple Sweep Feature later on.
First However I had to start with Headphone Jack. I Used simple scatch for that and Revolve Feature.
For this time I also used Extrude Feature after to make the cable for the jack connected
After that I could make a stress relief section using also Extrude but this time with Draft parameter set and unchecked Merge parameter.
That allowed me for making extruded cut after for making nice cuts from top to bottom without touching cable as its seperate body now.
When everything was prepared I could make an good use of 3d scatch and the cable profile to connect all togather using Sweep Feature.
At this point I learned about tangency relation and how to use it in 3d Scatch to make the connections smooth for the Sweep Feature.
According to what I managed to learn here I made a cable plug using same technique.
